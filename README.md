### Prerequisites

- Node.js 18+ and npm
- Git

### Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/RitaAkankunda/campus_hustle.git
   cd campus_hustle
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. **Start development server**

   ```bash
   npm run dev
   ```

5. **Open your browser**
   ```
   http://localhost:5173
   ```
